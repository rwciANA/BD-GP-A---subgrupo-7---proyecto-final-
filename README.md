# Sistema de Cooperativa de Ahorros

Este proyecto muestra una estructura base para conectar una base de datos PostgreSQL con un backend en Java Spring Boot y un frontend en React.

## Estructura

- backend/: API REST en Java
- frontend/: interfaz web en React

## Requisitos

- JDK 17 o superior
- Maven
- PostgreSQL
- Node.js 18 o superior

## Configuración de PostgreSQL

1. Crear una base de datos llamada `cooperativa`.
2. Ajustar las credenciales en `backend/src/main/resources/application.properties`.
3. Asegurarse de que exista una tabla llamada `clientes` o cambiar el entity a la tabla real.

## Ejecutar backend

```bash
cd backend
mvn spring-boot:run
```

## Ejecutar frontend

```bash
cd frontend
npm install
npm start
```
